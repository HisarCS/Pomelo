Pomelo

Pomelo is an interactive robot that teaches children basic algorithmic skills and enhances classroom collaboration. It becomes a part of the class instead of merely being a second teacher by interacting with children in the same way children interact with each other. It will encourage kids to use technology as an interactive learning tool instead of a form of addictive entertainment while creating a more cooperative and social classroom environment.

  • Helps the teacher in class
  • Improves the social skills of children
  • Teaches basic coding with coding blocks
  • Helps children catch up in class
